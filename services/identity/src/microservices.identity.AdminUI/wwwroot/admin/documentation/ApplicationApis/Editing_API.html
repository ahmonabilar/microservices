Editing an API

Access the API Details by selecting the API you want to edit in the APIs list.

You will notice that there are several key tabs, which we will cover in detail.

You will observe the tabs available to edit are Details, Capabilities and Properties.

Editing Details

On the Details tab, there is a display of the following:

  • Client ID
  • Display Names Documentation about internationalisation here.
  • Secret

Editing Capabilities

On the Capabilities tab, you can edit:

  • Allow Introspection
  • Custom Grant Types

If you have configured the 'CustomGrantTypes' settings, those options will show as suggested input in the Custom Grant Types field:

Editing Properties

On the Properties tab, you can add, modify and remove entries from the properties bag.

Saving and Deleting

The 'Save All' button at bottom of the page will be enabled with any valid change. When you are done and wish to keep those changes click the save button.

In this same part of the screen there is also a delete button for removing this API from the database.
